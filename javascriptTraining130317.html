Explicit and Implicit coercion

Using the

Number(..)

function is described as explicit coercion, converting a string to a number. If you are trying to compare a string to a number and consider them to be loosely equal you could use implicit coercion So if you use the

==

loose equals operator to make the comparison

"99.99" == 99.99

, JavaScript will convert the left-hand side

"99.99"

to its number equivalent

99.99

. The comparison then becomes

99.99 == 99.99

, which is of course true.

Loops

while and do while

'While' and 'do while' loops are designed to run a set of statements in a block until the action is no longer true. This is called an interation A 'while' loop will run until a condition is met but will not run if the condition is false to begin with. A 'do while' loop will run until the condition is met, it will run 1 iteration even if the result is false

Var numOfCustomers = 10

while (numOfCustomers > 0) {
console.log( "How may I help you?" );
numOfCustomers = numOfCustomers - 1;
}

do {
console.log( "How may I help you?" );
numOfCustomers = numOfCustomers - 1;
} while (numOfCustomers > 0);

//do loop prints to screen the question "How may I help you" //it runs on loop and then the while statement checks if the numOfCustomers is greater than 0 //this will run at least once regardless of numOfCustomers

for loop

"for" loops are used to run a set number of iterations and at variable increments The for loop has three clauses: the initialization clause (var i=0), the conditional test clause (i <= 9), and the update clause (i = i + 1). So if you're going to do counting with your loop iterations, for is a more compact and often easier form to understand and write.


for (var i = 0; i <= 9; i = i + 1) {
console.log( i );
}

Objects

An object holds properties and named locations that can be stored in a variable. It can be given any variable name. Object list items are seperated by a comma. To access the properties you can use dot notification by calling the variable name and the property, example obj.a

Var obj = {

a: "hello world",
b: 42,
c: true
};

obj.a;     //"hello world"

Alternatively you can use bracket notification to select an object property. This can be useful if the objects property name is stored in another variable and you can't select it through dot notification.

var obj = {
a: "hello world",
b: 42
};

var b ="a";

obj[b];     //"hello world"
obj["b"];     // 42
